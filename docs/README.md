All Vibe Flow documentation follows these standards:

### Front Matter Template
```yaml
---
title: Document Title
description: Brief description of the document
category: getting-started|guides|references|reports
version: 5.0
last_updated: YYYY-MM-DD
author: Vibe Flow Team
tags: [tag1, tag2, tag3]

```

### Structure Guidelines
1. **Clear Title**: Use H1 for the main title
2. **Logical Sections**: Use H2-H4 for sections
3. **Code Examples**: Include practical examples
4. **Bilingual Support**: Maintain English and Chinese versions
5. **Cross-references**: Link to related documents
6. **Version Tracking**: Note Vibe Flow version compatibility
